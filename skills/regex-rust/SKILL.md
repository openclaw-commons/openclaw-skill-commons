# Regex Rust

> 通用技能

## 📦 安装

```bash
npx clawhub@latest install regex-rust
```

## 🚀 快速开始

```bash
# 使用默认配置
openclaw regex-rust

# 查看帮助
openclaw regex-rust --help

# 指定配置
openclaw regex-rust --config ./config.yaml
```

## 📖 功能特性

- ✅ 完整的Regex Rust功能
- ✅ 开箱即用的默认配置
- ✅ 详细的文档和示例
- ✅ 社区支持和持续更新
- ✅ 与其他技能无缝集成

## 📝 使用示例

### 基础用法

```bash
openclaw regex-rust
```

### 高级用法

```bash
openclaw regex-rust --verbose --output ./output
```

### 与其他技能组合

```bash
openclaw regex-rust | openclaw task-decomposer
```

## ⚙️ 配置选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| `--verbose` | 详细输出模式 | `false` |
| `--output` | 输出目录 | `./output` |
| `--config` | 配置文件路径 | `./config.yaml` |
| `--help` | 显示帮助信息 | - |
